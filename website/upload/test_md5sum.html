File Upload hash

upload_md5sum.php checks the following parameters:
  • perChunkSums - 0 do not compute md5sums for the chunks 1 - compute md5sums for chunkSize blocks of data
  • chunkSize - size in bytes of the md5sum chunks
  • algo - specify the hash algorithm to use, see the list here
  • displayExecuteTime - display the time the PHP script took to execute if this parameter is set (regardless of value)
