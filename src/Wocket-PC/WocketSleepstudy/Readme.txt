This Project is written in Java for collecting data from Wocket sensors in their Bursty working mode.
In the Bursty working mode, Wockets send summary data (activity counts) and the last minute of raw data to PC every minute (if the connection is done).

To run the project, you need to:
1. Import the files into eclipse 

2. Set the Build path:
Right click on the project --> Build Path --> Add External Archives...
Then browse to the jar folder inside the project folder and select all the jar files.
